Git Bash Command

Email

To switch to root directory
cd ~

To delete a directory
rmdir directory or rm -rf directory

To start git 
git init inside the directory

To open VS Code from Gitbash
code .

To create file
touch filename 

To Add individual file in git
git add filename

To Add all files at same time
git add * or git add .

To commit file and directory
git commit -m "I add about"

To commit
small letter m i.e. commit -m "added assets file" 

main or master is where the Production projects recides

To change branch name
git branch -M main

To create new branch
git branch "branchname" i.e. git branch ikorodu-campus

To list created branch
git branch 

To change to another branch
git checkout "branchname" i.e. git checkout ikorodu-campus


To create a new branch and checkout at the same time
git checkout

To connect to remote repo i.e github repo 
git remote add origin "repo link from the github"  e.g git remote add origin https://GeneralOlutoye/MyReactProject.git

To upload local repo to remote
git push -u "branch name" e.g git push -u main
